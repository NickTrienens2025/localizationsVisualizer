About This Service

This service provides a modern web interface for managing and viewing Contentful localization files. It connects to your Contentful space and provides:

  • Real-time data from Contentful
  • Searchable and sortable tables
  • Section-based organization
  • JSON API for integration
  • Multi-language support (EN/FR)
API Endpoints
Data Endpoints:
  • /api/entries - Get all entries
  • /api/sections - Get all sections
  • /api/section/{id} - Get specific section
  • /api/localization - Get all localization files
Web Interface:
  • /table - Table view of entries
  • /sections - Table view of sections
  • /sections/view - Sections overview
  • /section/{id}/view - Section detail view
